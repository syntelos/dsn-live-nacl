
Code generation

    Fonts

        Generate ../FontFutural.{h,cc} from the "futural.jhf" embedded
        in the "fv3" package by running

         ./gen-font.sh

        Requires java

        Upstream http://fv3.googlecode.com/

    Shaders

        Generate ../Shaders.h from ../shaders/* by running

         ./gen-shaders.sh
